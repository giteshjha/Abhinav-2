# Modern Prince

**Modern Prince** is a high-fidelity, polished AI assistant and design forge. Built for the modern era to help you expand your kingdom of knowledge through text, image, and motion.

## Features
- **Royal Chat**: A polite and capable AI assistant powered by Gemini.
- **Image Forge**: Studio-quality image generation with "Magic Refine" prompt engineering.
- **Motion Studio**: Cinematic video generation powered by Veo.
- **Knowledge Kingdom**: Educational deep-dives into how Generative AI works.
- **PWA Ready**: Installable on Android and iOS as a native-feeling application.

## 🚀 How to Run Locally

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/[YOUR_USERNAME]/Abhinav.git
    cd Abhinav
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm run dev
    ```

## 📱 Mobile Installation (Android/iOS)
Once hosted on a secure (HTTPS) URL, open the link in your browser:
- **Android**: Tap ⋮ $\rightarrow$ "Install app".
- **iOS**: Tap the Share icon $\rightarrow$ "Add to Home Screen".

## 🛠 Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Motion (Animations)
- Google Gemini API (@google/genai)
- Lucide React (Icons)
